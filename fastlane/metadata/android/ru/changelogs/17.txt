## 0.9.6

1. Поддержка Fever API (бета)

2. Улучшения перевода (спасибо всем)

3. Исправлена ошибка синхронизации времени при переключении учетных записей

4. Исправлена ошибка, из-за которой вы не можете изменить другие параметры учетной записи

Устаревший Fever API, который не полностью совместим с современными читателями.
Но, как первый API сторонних сервисов, поддерживаемый службой Read You, это хорошая возможность начать работу.

В ближайшее время будет добавлено обновление для исправления ошибок и дополнительных переводов.


### Известные ошибки

1. Во время каждой синхронизации учетная запись Fever может извлекать только определенное количество статей на сервере

2. Функция оповещения учетной записи Fever не работает

3. После каждой синхронизации сбрасывается предварительно настроенный переключатель полного анализа содержимого учетной записи Fever
