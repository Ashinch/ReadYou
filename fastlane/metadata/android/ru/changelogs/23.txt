### Примечания

1. Спасибо сообществу FreshRSS, особенно @Alkarex, за помощь в этом проекте.
2. ReadYou в настоящее время не полностью совместим с Inoreader через API Google Reader.
3. ReadYou в настоящее время не может указать новую группу при добавлении подписки через API Google Reader.
4. При синхронизации с API Google Reader ReadYou может получать до 10 тысяч статей за одну синхронизацию. Для прочитанных статей будут синхронизированы только те, которые были прочитаны за последний месяц, в то время как непрочитанные и отмеченные статьи будут синхронизированы полностью. Скорость синхронизации в первую очередь зависит от количества этих статей. Первоначальная синхронизация может занять некоторое время, в зависимости от объема разницы данных между локальным устройством и сервером. Если у вас большое количество каналов, это может все еще казаться медленным, я попробую оптимизировать это, если потребуется.
5. Мы рекомендуем пользователям, которые участвовали в тестировании, повторно синхронизировать или повторно добавить свою учетную запись API Google Reader после установки версии `0.9.12`.
6. Будем с нетерпением ждать более подробных улучшений в следующей версии.
