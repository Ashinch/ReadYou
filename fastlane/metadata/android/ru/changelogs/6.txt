## Примечания

1. Каждый раз после холодного запуска проверяйте наличие обновлений автоматически (их можно будет использовать во время выхода следующей версии).

2. Во время проверки обновлений используйте Gitee API, если упрощенный китайский язык и GitHub API, когда другие.

3. Пожалуйста, не проверяйте обновления вручную слишком часто (установлен интервал 2 секунды), поскольку такой же IP может получить доступ только к GitHub API 60 раз в час, а точная граница частоты Gitee API неизвестна (она еще не была сброшена в течение 8 часов).

4. Рефакторинг прочитанных страниц появился после v0.8, а рефакторинг иконок и обложки появился после v0. 8, а поддержка FreshRSS (Fever API, Google Reader API) пришла после v1.0.

5. Эту версию можно синхронизировать только в одной ветке из-за изъяна приложения обработки upstream, который может быть немного медленным. Автор приложения упоминает, что он будет исправлен в следующей версии, и протокол Atom также будет поддерживаться.
