## 0.7.4

1. Tambahkan halaman "Kiat & Dukungan".

2. Tambahkan pemeriksaan pembaruan dan unduh.

3. Pilih tema warna dinamis secara default pada pemasangan pertama (jika tersedia).

4. Tambahkan fitur sinkronisasi gesek ke bawah ke halaman alur informasi.

5. Untuk sementara tambahkan petunjuk pemuatan saat penguraian konten lengkap.

6. Perbarui beberapa dependensi upstream (mungkin memiliki cacat yang tidak diketahui).

7. Mendukung protokol RSS v1.0 (untuk diuji).

8. Perbaiki latar belakang bilah navigasi sistem.

9. Perbaiki masalah tidak mencegat acara kembali saat menu opsi dibuka.

10. Optimalkan beberapa gaya video di halaman baca.

11. Optimalkan beberapa gaya abnormal.

### Catatan

1. Periksa pembaruan sekali secara otomatis setiap kali mulai dingin (Anda dapat mengalaminya saat versi berikutnya dirilis).

2. Saat memeriksa pembaruan, gunakan API Gitee untuk bahasa Cina Sederhana dan API GitHub untuk bahasa lain.

3. Harap jangan terlalu sering memeriksa pembaruan secara manual (interval pendinginan 2 detik telah ditetapkan), karena IP yang sama hanya dapat mengakses API GitHub 60 kali per jam, dan batas frekuensi pasti dari API Gitee tidak diketahui (belum disetel ulang selama 8 jam).

4. Refactoring halaman baca datang setelah v0.8, dan refactoring ikon dan sampul datang setelah v0.9, dan dukungan untuk FreshRSS (Fever API, Google Reader API) datang setelah v1.0.

5. Versi ini hanya dapat disinkronkan dalam satu utas karena bug plugin penguraian hulu, yang mungkin agak lambat. Pembuat plugin menyebutkan bahwa itu akan diperbaiki di versi berikutnya, dan protokol Atom juga akan didukung.
