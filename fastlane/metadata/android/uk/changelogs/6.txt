## Примітки

1. Щоразу після холодного запуску перевіряйте наявність оновлень автоматично (їх можна буде використовувати під час виходу наступної версії).

2. Під час перевірки оновлень використовуйте Gitee API, якщо спрощена китайська мова та GitHub API, коли інші.

3. Будь ласка, не перевіряйте оновлення вручну надто часто (встановлено інтервал 2 секунди), оскільки такий самий IP може отримати доступ лише до GitHub API 60 разів на годину, а точна межа частоти Gitee API невідома (вона ще не була скинуто протягом 8 годин).

4. Рефакторинг прочитаних сторінок з'явився після v0.8, а рефакторинг іконок та обкладинки з'явився після v0.8, а підтримка FreshRSS (Fever API, Google Reader API) прийшла після v1.0.

5. Цю версію можна синхронізувати лише в одній гілці через ваду застосунку обробки upstream, який може бути трохи повільним. Автор застосунку згадує, що він буде виправлений в наступній версії, і протокол Atom також буде підтримуватися.
