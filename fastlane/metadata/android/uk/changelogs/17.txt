## 0.9.6

1. Підтримка Fever API (бета)

2. Покращення перекладу (дякую всім)

3. Виправлено помилку синхронізації весь час під час перемикання облікових записів

4. Виправлено помилку, через яку ви не можете змінити інші параметри облікового запису

Fever — застарілий API, який не повністю сумісний з сучасними читачами.
Але, як перший API сторонніх сервісів, підтримуваний службою Read You, це гарна можливість почати роботу.

Найближчим часом буде додано оновлення для виправлення помилок та додаткових перекладів.


### Відомі помилки

1. Під час кожної синхронізації обліковий запис Fever може витягувати лише повну кількість статей на сервері

2. Функція сповіщення облікового запису Fever не працює

3. Після кожної синхронізації скидається попередньо налаштований перемикач повного аналізу вмісту облікового запису Fever
