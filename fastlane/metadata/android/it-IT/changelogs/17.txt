## 0.9.6

1. Aggiunto supporto alle API di Fever (beta)

2. Migliorate le traduzioni (grazie a tutti)

3. Sistemato il bug in cui l'app continua a sincronizzare quando si cambia account

4. Sistemato il bug che non ti permette di modificare le impostazioni degli altri account

Fever ha delle API datate che non sono pienamente compatibili con i moderni lettori.
Essendo tuttavia le prime API di un servizio di terzi ad essere supportato da Read You, è un buon inizio.

Ci saranno aggiornamenti aggiuntivi nel prossimo futuro per sistemare errori e ulteriori traduzioni.


### Errori noti

1. Gli account Fever possono ottenere il numero massimo di articoli sul server durante ogni sincronizzazione

2. Il servizio di notifiche per gli account Fever non funziona

3. L'intero parsing del contenuto degli account Fever è azzerato dopo ogni sincronizzazione
