## 0.9.6

1. Compatibilidad con la API de Fever (beta)

2. Traducciones mejoradas (gracias a los colaboradores)

3. Se corrigió el error de estar siempre sincronizado al cambiar de cuenta

4. Se corrigió un error que impedía cambiar la configuración de otra cuenta

Fever es una API en desuso, que no es compatible con los reproductores modernos.
Sin embargo, representa una buena oportunidad para comenzar, ya que es la primera API de servicios de terceros compatible con Read You.

Pronto estará disponible una actualización adicional para corregir errores y agregar nuevas traducciones.


### Errores conocidos

1. La cuenta Fever sincroniza todos los elementos del servidor cada vez que sincronizas

2. Las notificaciones no funcionan para las cuentas Fever

3. Los ajustes preestablecidos de escaneo de contenido para las cuentas de Fever se restablecen en cada sincronización
