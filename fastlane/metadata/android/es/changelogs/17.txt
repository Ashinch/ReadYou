## 0.9.6

1. API Fiebre de soporte (beta)

2. Traducciones mejoradas (gracias a todos)

3. Solucione el problema de que la cuenta siempre está sincronizada al cambiar de cuenta

4. Se solucionó el problema de que no se podían modificar otras configuraciones de la cuenta

Fever es una API obsoleta que no es totalmente compatible con las capacidades de los lectores modernos.
Pero como la primera API de servicio de terceros admitida por Read You, es una buena oportunidad para comenzar.

Habrá una actualización complementaria en un futuro cercano para corregir errores y traducciones adicionales.


### Problemas conocidos

1. La cuenta de Fever solo puede extraer todos los artículos del servidor cada vez que se sincroniza

2. La función de notificación de la cuenta Fever no es válida

3. El interruptor predeterminado de análisis de texto completo de la cuenta Fever se restablece después de cada sincronización
